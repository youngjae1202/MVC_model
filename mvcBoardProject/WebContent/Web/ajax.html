JQuery Ajax를 이용한 서버페이지 로딩

Ajax를 편리하게 사용하는 Jquery Ajax 를 이용하여 서버에 있는 페이지를 로딩

  1. AJAX는 에이잭스라 발음함
  2. AJAX는 Asynchronous JavaScript and XML의 약어임
  3. AJAX는 서버와의 데이터 교환을 비동기적으로 Background에서 페이지를 Reloading하지 않고 Update하는 기술로 Gmail, Google Maps, Youtube, and Facebook tabs에서 활용하고 있음
  4. JQuery없이 AJAX를 구현하는 것은 꽤 까다로움
