Fading Effect

jquery는 선택된 엘리먼트 투명도를 점진적으로 변경시키는 4개의 메서드를 가지고 있음

  1. fadeTo(speed, opacity, callback) :  불투명도 opacity로 점진적으로 투명도 변경
  2. fadeIn(speed, callback) :  opacity를 0에서 1로 변경(show)
  3. fadeOut(speed, callback) : opacity를 1에서 0으로 변경(hide)
  4. fadeToggle(speed, callback) : 반복
파라메터 speed는 fade처리 시간, opacity는 불투명도(0~1),callback은 콜백 함수 정의이며 모든 파라메터는 옵션 파라메터임

Code Sample

$(document).ready(function(){
	$("#lnkFadeOut").click(function(){
		$("#imgFade").fadeOut(2000);
	});
	$("#lnkFadeIn").click(function(){
		$("#imgFade").fadeIn(2000);
	});
	$("#lnkFadeToggle").click(function(){
		$("#imgFade").fadeToggle(2000);
	});	
});
