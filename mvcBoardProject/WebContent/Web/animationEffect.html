Animation Effect

Jquery는 애니메이션 방법을 제공하고 관련 메서드 형식은 다음과 같다
animate( properties, [duration], [easing], [callback])

  1. properties: 애니메이션을 통해 변경될 css의 속성을 나타내는 파라메터 css의 속성 표시방법으로 작성 {width : 200px;} 하며 필수 항목임
  2. duration: 애니메이션에 걸리는 시간으로 옵션 항목이며 밀리세컨드나 slow,fast로 표시
  3. easing: 변경에 사용되는 수학적 함수의 이름으로 옵션 사항
  4. callback: 애니메이션 완료후 실행할 함수로 옵션 사항

Example


애니메이션실행

이영역은 애니메이션실행을 클릭시 보여주는 영역으로 애니메이션실행 클릭전에는 보이지 않게 하고 클릭하면 크기를 변경하는 animate()메서드로 보이게 되는 영역

원래크기로 복귀

Code Sample

$(document).ready(function(){
	$("#lnkMoreDetail").click(function(){
		$("#img1").animate({width: 449, height: 339},"slow");
		//css에서는 크기 단위 px을 붙이나 jquery에서는 안붙임
		$("#imgDescription").show();
		$("#lnkMoreDetail").hide();
	});	
	
	$("#lnkClose").click(function(){
		$("#imgDescription").hide();
		$("#img1").animate({ height: 120 }, "fast");
        $("#img1").animate({ width: 120 }, "fast");
        $("#lnkMoreDetail").show();
	});
});
